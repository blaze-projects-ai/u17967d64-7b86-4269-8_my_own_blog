# My Own Blog

A simple blog application with posts, categories, and a clean reading experience.

## Features

- Homepage listing posts with titles, dates, and excerpts
- Individual post pages with full content and clean typography
- Category filtering
- 4 sample posts across 4 categories (Technology, Food, Lifestyle, Design)
- Responsive design with readable font sizes, proper line height, and max-width content

## Setup

```bash
npm install
npm start
```

Visit `http://localhost:3000`.

## Testing

```bash
npm test
```
